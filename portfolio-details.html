Project Details

Data visualization using Amazon S3 and Amazon Quicksight

Project information

  • Category: AWS project
  • AWS Used services: Amazon S3 , Amazon Quicksight
  • Project date: 01 October,2024
  • Dataset website: www.brightdata.com

Objective

Created Data Amazon visualization using Amazon S3 and Amazon Quicksight, Working with large Dataset of Amazon products global.

Amazon web services that used :

  • Amazon S3 (simple service storage )
  • Amazon Quicksight

    • Amazon Quicksight : Serverless machine learning-powered business intelligence Services to create interactive dashboards.
    It's fast,automatically scalable,embeddable,with per-session pricing.Also ,it's integrate with RDS,Aurora,Athena ,Redshift,S3...
    Amazon S3 :Amazon Simple Storage Service (Amazon S3) is an object storage service offering industry-leading scalability, data availability, security, and performance. ,Amazon S3 allows people to store objects (files) in "buckets" (directories),the buckets must have a globally unique name
